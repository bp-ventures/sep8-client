Stellar SEP-8 Client Webapp built using Next.js.

### Development

Create `.env.local`:

```
NEXT_PUBLIC_APPROVAL_SERVER_URL="https://as-uat.bpventures.us"
```

Install dependencies:

```
asdf install
npm install
```

Run dev server:

```
npm run dev
```

### Production

Create `.env.local`:

```
NEXT_PUBLIC_APPROVAL_SERVER_URL="https://as-uat.bpventures.us"
```

Install dependencies:

```
asdf install
npm install
```

Run production server:

```
npm run build
npm start
```
